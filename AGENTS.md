# AGENTS.md

This repository is a GitHub Pages website for showcasing the TaDaS project:
**Text as Data as Survey**.

Use `draft/OpenEcon.tex` as the source of truth for the project framing,
figures, terminology, and claims. The website should present the project clearly
for visitors, not reproduce the full paper.

Keep the site:

- Short, polished, and easy to scan.
- Focused on the question: are economists open to AI?
- Clear about the method: TaDaS turns naturally occurring text into
  survey-like evidence through cross-corpus semantic retrieval.
- Grounded in the application: EJMR discussion plus elite economics and finance
  publications.
- Suitable for static hosting on GitHub Pages.

Avoid overstating causal claims. The draft frames the evidence as descriptive:
AI-related professional discussion is tense, but less negative when AI is more
visible in elite-journal research.
